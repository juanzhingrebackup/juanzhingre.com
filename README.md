## Getting Started Guide

1. Clone
    ```bash
    git clone https://github.com/byjohnmichael/juanzhingre.com.git
    ```
2. Navigate
    ```bash
    cd juanzhingre.com
    ```
3. Installation
    ```bash
    npm install
    ```
4. Develop
    ```bash
    npm run dev
    ```
5. Lint
    ```bash
    npm run lint
    ```
6. Format
    ```bash
    npm run format:check
    npm run format
    ```
7. Build
    ```bash
    npm run build
    ```
8. Start
   ```bash
   npm run start
   ```
